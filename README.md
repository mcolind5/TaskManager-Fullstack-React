## How to Run

**Backend** (IntelliJ):
```bash
cd backend && mvn spring-boot:run
```

**Frontend** (VSCode):
```bash
cd frontend && npm install && npm run dev
```

## Folder Structure
```
backend/     # Java, Spring Boot
frontend/    # TypeScript
```
